# OpenCode One-Click Setup

Download `setup.bat`, double-click it, and everything installs automatically.

## How to use

1. Download **setup.bat**
2. **Double-click** it
3. Wait for it to finish
4. OpenCode launches automatically

## What it installs

- **Node.js** (via winget)
- **OpenCode** (via npm)
- **Ollama** (checks if installed)
- **AI Model** (deepseek-coder:6.7b)

## Requirements

- Windows 10 or 11
- 8GB RAM minimum
- 5GB free storage
- [Ollama](https://ollama.com/download/windows) (install it first, or it'll open the download page)
