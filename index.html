OpenCode One-Click Setup

Free AI coding assistant for your terminal. One download, zero config.

Download & Install

Choose your operating system:

Windows (.bat) macOS (.sh) Linux (.sh)
Windows: download setup.bat, double-click it, wait for it to finish.
macOS/Linux: download the .sh file, open terminal, run chmod +x setup-*.sh && ./setup-*.sh

What Happens After Install

Once setup finishes, OpenCode opens automatically. You'll see a terminal interface.

  • First launch — Press any key, select Ollama as provider, choose deepseek-coder:6.7b
  • Ask something — Type what you want, like Create a calculator in Python
  • Let it work — OpenCode reads, writes, and edits files in your project

How to Open OpenCode Again

After the first run, just type this in any terminal:

opencode

Runs in whatever folder you're in. Point it at your project:

cd C:\Users\ASUS\Desktop\my-project
opencode

Commands Inside OpenCode

  • /models — Switch AI model
  • /undo — Undo last change
  • /help — Show all commands
  • Ctrl+C — Exit

Requirements

  • Windows 10/11, macOS, or Linux
  • 8GB RAM minimum (16GB recommended)
  • 5GB free storage for AI model
  • Ollama installed (setup will prompt you if missing)
