⬇ Option 1 — Auto Installer (Windows)

Download and double-click:

Download install-opencode-windows.bat

It installs Node.js, OpenCode, Ollama, AI model, and launches OpenCode automatically.

⚙ Option 2 — Manual (All systems)

Open PowerShell and follow:

1
Install OpenCode
npm install -g opencode-ai
2
Install Ollama
Download from ollama.com/download/windows
3
Download AI model
ollama pull deepseek-coder:6.7b
4
Launch OpenCode
opencode

🔄 Open OpenCode again

Any time, just type:

opencode

For a specific project:

cd Desktop\my-project
opencode

⌨ Commands inside OpenCode

/modelsSwitch AI model
/undoUndo last change
/helpShow all commands
Ctrl+CExit
