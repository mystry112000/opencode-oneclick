⬇ Download & Install

Choose your operating system. The installer handles everything — Node.js, OpenCode, Ollama, and the AI model.

Windows (.bat) → macOS (.sh) → Linux (.sh) →
Windows: download setup.bat, double-click it, wait for installer to finish.
macOS / Linux: download .sh, open terminal, run chmod +x setup-*.sh && ./setup-*.sh

🚀 After the Installer Finishes

OpenCode opens automatically. Here's what to do next:

  • 1 Press any key to continue
  • 2 Use arrow keys to select Ollama as provider → press Enter
  • 3 Select deepseek-coder:6.7b → press Enter
  • 4 Type what you want, for example: Create a calculator in Python

🔄 How to Open OpenCode Again

After the first run, just type this in PowerShell or CMD from any folder:

opencode

Point it at your project folder:

cd Desktop\my-project
opencode

❓ Frequently Asked Questions

Do I need internet?
Yes, to download the AI model (~4GB) on first run. After that, everything runs offline on your computer.
Is it really free?
100% free. No API keys, no subscriptions, no token limits. OpenCode uses Ollama which runs locally on your machine.
Can I use it with other AI models?
Yes. Inside OpenCode, type /models to switch between any Ollama model you've downloaded.
Does it send my code anywhere?
No. Everything runs locally. Your files never leave your computer.
It's slow. What can I do?
Use a smaller model. Run ollama pull llama3.2 for faster responses, or upgrade to 16GB+ RAM for larger models.

📋 Requirements

  • ✓ Windows 10/11, macOS 12+, or Linux
  • ✓ 8GB RAM minimum (16GB recommended)
  • ✓ ~5GB free storage for the AI model
  • ✓ Ollama (the script will prompt you if missing)
