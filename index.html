⬇ Option 1 — Auto Installer (Windows)

Download and double-click. It installs everything and launches OpenCode.

Download install-opencode-windows.bat

Installs: Node.js → OpenCode → Ollama → AI Model → Launch

⚙ Option 2 — Manual (Works on all systems)

If the auto installer doesn't work, follow these 4 steps:

1
Install OpenCode
npm install -g opencode-ai
2
Install Ollama
Download: ollama.com/download/windows
3
Download AI model
ollama pull deepseek-coder:6.7b
4
Launch
opencode

🔄 How to Open OpenCode Again

Anytime, just type:

opencode

For a specific project:

cd Desktop\my-project
